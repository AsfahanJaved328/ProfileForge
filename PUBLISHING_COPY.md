## LinkedIn Post Draft

I recently built **ProfileForge**, a local profile-writing project in **Python** and **PyTorch**.

The goal was to understand how a small text model can be trained from scratch for a focused use case instead of just using general text generation. I structured the project around professional writing such as profile headlines, about sections, and project bullets.

What the project includes:

- custom training data for profile-style writing
- checkpoint-based training and resume support
- structured generation for different profile sections
- local execution on CPU
- a demo workflow and documentation

This project helped me strengthen my understanding of:

- PyTorch model training
- text generation workflows
- debugging and iteration on Windows
- organizing a project for presentation and reuse

GitHub: `https://github.com/AsfahanJaved328/ProfileForge`
