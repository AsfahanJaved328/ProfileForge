# LinkedIn Post

I recently completed **ProfileForge**, a personal project built with **Python** and **PyTorch**.

ProfileForge is a local profile-writing text generation project designed to explore how a small transformer model can be trained for a focused use case such as:

- LinkedIn headlines
- about sections
- project bullets
- profile summaries

Instead of treating text generation as a general black box, I wanted to understand the full workflow more directly. This project gave me hands-on experience with:

- dataset preparation
- checkpoint-based training
- resume logic for longer runs
- structured prompt generation
- debugging and iteration on Windows

I also shaped the project around professional writing so it could support portfolio and profile presentation, not just raw text sampling.
